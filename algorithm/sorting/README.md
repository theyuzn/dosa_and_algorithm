# Sorting

Sorting in C

### File Structure
```sh
sorting/
    │
    ├── inc/
    |   ├── search.h    # Binary search and linear search
    │   ├── merge.h     # Merge sort
    │   ├── heap.h      # Heap sort
    │   ├── bubble.h    # Bubble sort
    │   └── quick.h     # Quick sort
    │
    ├── src/
    |   ├── search.c
    │   ├── merge.c
    │   ├── heap.c
    │   ├── bubble.c
    │   └── quick.c
    │
    └── main.c
```
